# User Service

## Description

This handles Authentication and User Related Requests

## Installation

```bash
$ npm install
```

## Running the app

From the Root directory

```bash
# development
$ npm run start user-service

# watch mode
$ npm run start:dev user-service

# production mode
$ npm run start:prod user-service
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
