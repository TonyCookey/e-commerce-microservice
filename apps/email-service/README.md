# Email Service

## Description

This handles sending emails to newly registered users.

## Installation

```bash
$ npm install
```

## Running the app

From the Root directory

```bash
# development
$ npm run start email-service

# watch mode
$ npm run start:dev email-service

# production mode
$ npm run start:prod email-service
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
